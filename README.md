## Instalación

```bash
$ git clone https://gitlab.com/tfg5261618/beatmarket.git
$ cd beatmarket
$ npm install
$ npx ionic build
$ npx ionic cap add android
$ npx ionic cap add ios
```

Para ejecutar un servidor local de la aplicación en el navegador:

```bash
$ npx ionic serve
```
